# TaskFlow — Sistema Gerenciador de Tarefas Online

Sistema web desenvolvido como projeto extensionista em parceria com a M&A Soluções Financeiras.

## Como usar

1. Abra `sistema_web.html` no navegador
2. Login: `admin@masf.com.br` / `123456`

## Funcionalidades

- Autenticação de usuários (login e cadastro)
- Criação, edição, exclusão e gerenciamento de projetos
- Criação, edição, exclusão, atribuição e acompanhamento de tarefas
- Quadro Kanban com drag-and-drop
- Calendário com prazos
- Dashboard com indicadores

## Tecnologias

HTML5 · CSS3 · JavaScript
